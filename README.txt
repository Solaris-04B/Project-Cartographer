Overview
---
PROJECT ARTEMIS - A fan-made, free-to-play shooter set in the Halo universe. PROJECT ARTEMIS is being developed in Unity 3d (Version 5.5.0f3) personal edition, using a combination of software and plugins (Will be listed below). PROJECT ARTEMIS focuses on open-world exploration and a combination of campaign and firefight currently called "Conquest", the project will also feature Forge as well as War Games, although much later down the road.
Game Content Usage Rules: Halo© Microsoft Corporation. PROJECT ARTEMIS was created under Microsoft's "Game Content Usage Rules" using assets from Halo, and it is not endorsed by or affiliated with Microsoft.  

Game Design Style - Bungie Era (Reach/ODST/Halo Wars 2)
